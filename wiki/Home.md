# Cosmic Almanac Wiki

Cosmic Almanac 是一个轻量级、双语、可部署到 GitHub Pages 的每日神谕网页应用。用户输入一次生日后，应用会在本地浏览器中根据出生日期和当前日期生成稳定的每日解读。

## 当前状态

V1 已经完成初步闭环：

- 单页静态应用，无后端、无构建工具、无依赖安装。
- 支持 English 和简体中文。
- 支持夜晚 / 白天主题。
- 支持 PWA manifest 和 iPhone 主屏幕添加。
- 支持 GitHub Pages 托管。
- 已补齐基础 Open Graph / Twitter Card 元数据，便于分享应用链接时显示标题和描述。
- 当前正式名称为 **Cosmic Almanac**。

## 核心产品一句话

输入生日，获得一份结合西方星座、中国生肖、月相、月入星座、行星日和每日宜忌的双语每日解读。
