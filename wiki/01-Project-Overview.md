# 项目总览

## 产品定位

Cosmic Almanac 是一个双语每日神谕小应用。它不是专业星历工具，也不是医学、法律、金融或科学建议系统；它更接近一个文化化、象征化、轻量化的每日仪式工具。

用户只需要输入一次生日。之后每天打开应用，页面会生成一份稳定的每日解读，包括身份标签、今日天象、今日宜忌、四个生活维度评分、今日神谕和宇宙建议。

## V1 目标

V1 的目标不是做复杂账号系统或后端服务，而是完成一个可分享、可安装、可长期打开的静态网页应用：

- 用户能直接打开页面使用。
- 页面在手机上像 App 一样清爽。
- 每日结果稳定，不会刷新一次变一次。
- 内容足够丰富，短期使用不容易重复。
- 项目可以直接部署到 GitHub Pages。
- 代码结构足够简单，适合继续迭代。

## 当前主要功能

- 生日输入和本地保存。
- 西方星座计算。
- 中国生肖计算。
- 月相计算。
- 月入星座估算。
- 星期行星日映射。
- 今日宜忌生成。
- 四类评分：事业职场、感情人际、身体健康、财运偏财。
- 今日神谕和宇宙建议。
- 卡片点击展开详情弹窗。
- 日期前后切换和回到今天。
- 中英文切换。
- 白天 / 夜晚主题切换。
- PWA manifest 和 App 图标。
- 链接分享标题与描述元数据。


## 为什么保持静态应用

当前项目刻意不引入框架、构建工具、后端或数据库。这样做的好处是：

- GitHub Pages 可以直接部署。
- 维护成本低。
- 用户隐私更简单，生日只保存在本地浏览器。
- 后续迁移到 React、Vue 或后端服务时，现有规则和内容仍然可以复用。

## V1 成熟度判断

V1 已经具备完整产品闭环：输入、保存、每日生成、解释、切换语言、切换主题、安装、分享、部署文档都已覆盖。后续迭代的重点不再是补基础功能，而是提高内容深度、视觉包装和长期使用体验。
