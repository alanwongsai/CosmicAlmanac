# V1 版本说明

## 版本状态

Cosmic Almanac V1 是当前第一个完整可用版本。它已经从早期单页原型发展成一个完整的双语每日神谕 PWA。

当前远端主分支最新版本记录为 `1.0.2`。

## V1 已完成内容

### 产品体验

- 首次打开时显示生日输入页。
- 输入生日后进入每日解读页。
- 再次打开时自动读取本地生日并直接显示今日解读。
- 支持切换前一天、后一天和回到今天。
- 大多数信息卡片可以点击，打开居中详情弹窗。
- 日夜模式和语言按钮已改为随页面滚动，不再固定遮挡内容。

### 内容系统

- English 和简体中文双语内容。
- 12 个西方星座。
- 12 个中国生肖。
- 8 个月相。
- 12 个月入星座。
- 7 个行星日。
- 90 条今日神谕。
- 90 条宇宙建议。
- 四个评分维度，每个维度 5 个分数档，每档 12 条短句。
- 今日宜忌由五个因素组合生成，组合空间为 96,768。

### 技术能力

- 无后端。
- 无构建工具。
- 无第三方依赖。
- 使用 `localStorage` 保存生日、语言和主题偏好。
- 使用 seeded pseudo-random generator 保证同一天结果稳定。
- 使用 Julian date 相关计算生成月相。
- 使用月亮平均黄经估算月入星座。
- 使用 Open Graph 和 Twitter Card 元数据优化链接分享。

### PWA 和部署

- `manifest.json` 已配置应用名称、启动页、显示模式、主题色和图标。
- 支持添加到 iPhone 主屏幕。
- 支持 GitHub Pages 静态部署。
- 当前应用正式名称为 **Cosmic Almanac**。

## V1 不包含的内容

- 不包含账号系统。
- 不包含云端同步。
- 不包含后端 API。
- 不包含支付、订阅或登录。
- 不包含专业天文星历精度。
- 不包含严格农历新年边界的生肖计算。
- 不包含完整社交分享图，当前只提供标题和描述元数据。

## V1 的定位

V1 是一个完整可用的静态产品版本。它适合公开分享、部署到 GitHub Pages、添加到手机主屏幕，也适合作为后续扩展内容库、视觉包装和产品化功能的基础。
